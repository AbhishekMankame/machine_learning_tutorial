## K-Nearest Neighbors (KNN) Algorithm

### What is KNN?
K-Nearest Neighors (KNN) is a supervised machine learning algorithm used for both classification and regression tasks.
It works by finding the K closest data points (neighbors) to a new data point and making a prediction based on those neighors.
- Classification: Predicts a category (e.g., Spam or Not Spam).
- Regression: Predicts a numerical value (e.g., house price).

### Definition
K-Nearest Neighbors (KNN) is a non-parametric, instance-based (lazy learning) algorithm that classifies or predicts a value based on the majority class (classification) or average value (regression) of the K nearest training examples.

### Why it is called "Lazy Learning"?
Unline algorithms like Decision Trees or Neural Networks, KNN does not build a model during training.
Instead:
- It simply stores the training data.
- When a new sample arrives, it calculates the distance to every stored sample.
- It predicts using the nearest neighbors.

### Distance Metrics
1. Euclidean Distance ---> d = sqrt((x2-x1)^2 + (y2-y1)^2)
2. Manhattan Distance ---> |x2 - x1| + |y2 - y1|

### Choosing K
Small K: K=1
Advantages:
- Learns complex patterns
Disadvantages:
- Sensitive to noise
- Can overfit

Large K: K = 15
Advantages:
- Stable
- Less affected by noise
- Misses local patterns

A common starting point is: K==sqrt(N), where N is the number of training sample

### Python Code (Without Library)

import math 
training_data = [
    ([170, 65], "Fit"),
    ([175, 70], "Fit"),
    ([160, 55], "Fit"),
    ([180, 90], "Not Fit"),
    ([185, 95], "Not Fit")
]
new_point = [178, 75]
k = 3
distances = []

for point, label in training_data:
  distance = math.sqrt(
    (point[0]-new_point[0])**2 + (point[1]-new_point[1])**2
  )
  distances.append((distance, label))

distances.sort()
neighbors = distances[:k]
votes = {}

for d, label in neighbors:
  votes[label] = votes.get(label, 0) + 1

prediction = max(votes, key=votes.get)
print("Prediction:", prediction)


### Python Code Using Scikit-learn
from sklearn.neighbors import KNeighborsClassifier
#### Features
X = [
    [170,65],
    [175,70],
    [160,55],
    [180,90],
    [185,95]
]

#### Labels
y = [
    "Fit",
    "Fit",
    "Fit",
    "Not Fit",
    "Not Fit"
]

#### Create model
model = KNeighborsClassifier(n_neighbors=3)

#### Train model
model.fit(X,y)

#### Prediction
prediction = model.predict([[178, 75]])
print(prediction)

### Advantages:
- Very simple to understand and implement
- No training phase (lazy learning)
- Can solve both classification and regression problems
- Works well with small datasets
- Can model complex decision boundaries

### Disadvantages
- Slow predictions for large datasets because it compares the new point with all training samples.
- Requires lot of memory to store the training data.
- Sensitive to irrelevant features.
- Sensitive to different feature scales (feature scaling is usually needed).
- Performance decreases in high-dimensional data (the "curse of dimensionality").

### Applications
- Image classification
- Handwriting recognition
- Recommendation systems
- Medical diagnosis
- Credit risk analysis
- Fraud detection
- Face recognition
- Customer segmentation

### Time Complexityu:
Let:
    N = Number of training samples
    D = Number of features

Training --> O(1) stores the data
Distance calculation --> O(N*D)
Sorting distances --> O(N log N)
Prediction --> Approximately O(N*D) (or O(N log N) if sorting is used)

### Summary
Feature: K-NN
Learning Type: Supervised
Algorithm Type: Lazy Learning
Uses Training Phase: No (stores data only)
Tasks: Classification and Regression
Prediction Method: Majority vote (classification)/ Average(regression)
Distance Measures: Euclidean, Manhattan, Minkowski, Hamming
Sensitive to Feature Scaling: Yes
Main Hyperparameter: K (number of neighbors)
Best for: Small to medium-sized datasets with meaniningful distance relationships
